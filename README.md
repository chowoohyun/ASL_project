# ASL_project

- 메타버스 아카데미의 프로젝트

- 수화를 이미지 분류를 통해 인식하는 프로젝트 전이학습과 데이터 증강 학습을 위해 진행

- 다양한 전이학습을 진행하였지만, 제일 성능이 좋은 vgg16을 사용하기로 결정

- 데이터 증강과 전이학습에 대한 이해도를 높히는 프로젝트, 추후 yolo를 사용해서 실시간 이미지 처리에도 적용해볼만한 프로젝트
